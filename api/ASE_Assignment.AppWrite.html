Remarks

This method ensures that the parameter list contains exactly one item and that the text is not null or empty.

CommandException

Thrown when the parameter list does not contain exactly one item.

Execute()

Executes the command by writing the specified text to the canvas. It retrieves the text to be written from the command's parameters and passes it to the canvas' WriteText(string) method to display it.

public override void Execute()

Remarks

This method retrieves the text parameter from the command and calls the WriteText(string) method to display the text on the canvas.
